The key scripts can all be found in the Assets folder.

- Heap.cs holds the API methods themselves (Read(), Write(), Initialise(), Free() etc)
- Constants.cs, Molecule.cs and Session.cs are support classes for the API.
- Main.cs and Tests.cs are test code demonstrating use and durability of the code.
- JellyfishXMLgui.cs is the Pocket Spacecraft full program example which reads in 
custom XML, stores it in an Ocean and then saves it back into an XML file.

To run an code in Unity, open a new project and replace the Assets and ProjectSettings folders
with those included here.
